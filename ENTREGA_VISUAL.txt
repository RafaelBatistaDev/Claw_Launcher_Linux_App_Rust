┌─ FLUXO DE USO ──────────────────────────────────────────────────────────────┐

│      Binário criado em: ~/.local/bin/claw-launcher                         │
│                                                                              │
│  2️⃣  CRIAR INSTÂNCIA (automático)                                           │

│      Menu KDE → Procura → Seleciona app                                    │
│      ↓                                                                       │

│      ↓                                                                       │
